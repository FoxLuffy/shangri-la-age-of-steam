# Story Richness Roadmap: Shangri-la: Age of Steam

## Phase 1: Core Foundations (Completed)
- [x] **The Steam Ethos**: Basic "Industrialism vs Nature" atmosphere.
- [x] **Reactive Dialogue**: Prompt-based reactive NPC responses.
- [x] **Memory Persistence**: Foundation for long-term NPC memories.

## Phase 2: Narrative Agency (Completed)
- [x] **Dynamic Mood System**: Player mood influencing narrative tone.
- [x] **Memory Persistence**: NPCs remember specific interactions.
- [x] **World-State Memory**: Shared knowledge of past events.

## Phase 3: Interactional Complexity (Completed)
- [x] **Multi-Agent Orchestration**: NPCs interacting with each other.
- [x] **Persistent History**: Tracking global narrative beats.
- [x] **Branching Choice Consequences**: Basic narrative outcome branching.

## Phase 4: Deep World-Building (Ongoing)
- [x] **Dynamic Inventory & Quests**: Items as plot points.
- [x] **Faction & Allegiance System**: NPC loyalties to specific industrial guilds.
- [x] **Procedural Plot Hooks**: Generating new quests based on current world state.

## Phase 5: Atmospheric Richness (Next)
- [ ] **Environmental Storytelling**: Description of "decay" and "progress" in specific districts.
- [ ] **Faction-Specific Dialogue**: NPCs using different dialects or slang based on guild affiliation.
- [ ] **Dynamic World Events**: Narratively significant events (e.g., a factory strike or a boiler explosion).

## Phase 6: The Human Element (Future)
- [ ] **Legacy & Reputation**: Permanent changes to the world based on player industrial influence.
- [ ] **Character Origin System**: Distinct starting backgrounds for players.
- [ ] **Relationship Dynamics**: Deepening NPC-to-NPC history and long-term emotional arcs.

## Phase 7: Sensory Narrative (Future)
- [ ] **Narrative Music Cues**: Dynamic scores for specific narrative beats.
- [ ] **Spatial Soundscapes**: Spatial audio for workshop/street environments.
- [ ] **Visual Narrative cues**: Visual descriptions of steam intensity and light.
