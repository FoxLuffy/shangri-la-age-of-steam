# Web Frontend Shell (React/Vite) Implementation Plan

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Set up a React frontend using Vite to interact with the FastAPI backend.

**Architecture:** A Vite-powered React SPA with a clean terminal-style UI, connecting to the backend via `fetch` or `axios`.

**Tech Stack:** React, Vite, Tailwind CSS, Axios

---

### Task 1: Initialize Frontend Repository

**Objective:** Create a Vite + React + Tailwind CSS project.

**Files:**
- Create: `shangri-la-age-of-steam/frontend/`

**Step 1: Create project with Vite**
Run: `cd /opt/data/workspace/saos/shangri-la-age-of-steam && npm create vite@latest frontend -- --template react-ts`
*Note: This requires `npm` or `corepack`. If not available, I'll use `uv` to run a script.*

**Step 2: Install Tailwind CSS**
Run: `cd frontend && npm install -D tailwindcss postcss autoprefixer && npx tailwindcss init -p`

**Step 3: Configure Tailwind**
Modify `frontend/tailwind.config.js`.

**Step 4: Commit**
```bash
git add frontend/
git commit -m "chore: initialize react frontend"
```

### Task 2: Create API Client & Basic Layout

**Objective:** Create a basic layout and a hook/service to fetch the health check.

**Files:**
- Create: `shangri-la-age-of-steam/frontend/src/api.ts`
- Create: `shangri-la-age-of-steam/frontend/src/App.tsx`

**Step 1: Implement API Client**
```typescript
import axios from 'axios';

const api = axios.create({
  baseURL: 'http://localhost:8000',
});

export const fetchHealth = async () => {
  const { data } = await api.get('/health');
  return data;
};
```

**Step 2: Implement App.tsx**
Create a basic UI with a "Check Backend Status" button.

**Step 3: Run and Verify**
Run: `cd frontend && npm run dev`
Verify: `http://localhost:5173` shows "Status: OK".

**Step 4: Commit**
```bash
git add frontend/
git commit -m "feat: add basic layout and health check"
```
