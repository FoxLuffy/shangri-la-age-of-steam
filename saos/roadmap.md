# Roadmap - Shangri-la: Age of Steam

## Phase 1: Foundation & Core API
- [ ] **Backend Environment:** Setup FastAPI server and vLLM connectivity.
- [ ] **State Management:** Define core JSON schema for "World State".
- [ ] **Basic API:** Create `/chat` endpoint for raw prompt exchange.

## Phase 2: Frontend Shell
- [ ] **Web Server:** Setup Vite + React environment.
- [ ] **UI Layout:** Create basic terminal-style UI with input field.
- [ ] **API Connection:** Connect frontend to Backend `/chat` endpoint.

## Phase 3: AI Narrative Logic
- [ ] **Prompt Engineering:** Develop system prompts for NPC and Narrator.
- [ ] **Context Injection:** Implement "Memory" system for NPC interactions.
- [ ] **Dynamic Effects:** Add ability for AI to trigger world events.

## Phase 4: Polish & Deployment
- [ ] **Persistence:** Add SQLite for saving games.
- [ ] **Asset Integration:** Add stylized CSS/animations.
- [ ] **Deployment:** Dockerize the whole suite.
